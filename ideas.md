# MemeBot - Анализ и идеи для новой реализации

## Анализ оригинального MemeBot (Kotlin)

### Алгоритм работы

1. TDLib авторизуется по номеру телефона
2. Приходят сообщения из отслеживаемых каналов (FAVORITE)
3. Для каждого сообщения с медиа:
   - Извлекается `minithumbnail` (встроенный в TDLib объект ~32x32 JPEG)
   - Вычисляется MD5-хеш байтов миниатюры
   - Проверяется в базе на дубликат
   - Если новое → сохраняется + пересылается в proxy-канал
   - Если дубликат → обновляется счётчик источников
   - Если ≥3 источников → пересылается в filtered-канал ("вирусный" мем)

### Механизм сравнения thumbnail

| Параметр | Значение |
|----------|----------|
| Метод | Точное совпадение MD5 |
| Входные данные | JPEG-байты minithumbnail от Telegram |
| Fuzzy matching | **Нет** |
| Perceptual hash | **Нет** |

### Почему это работает

Telegram генерирует `minithumbnail` детерминированно — для одинакового загруженного медиа minithumbnail идентичен. MD5 даёт быстрый O(1) lookup.

### Что ловит текущий алгоритм

- **Репосты** и **форварды** одного сообщения
- Копии через "сохранить → переслать в тот же Telegram"

### Чего НЕ ловит

- Независимые загрузки одного мема с диска разными пользователями
- Скриншоты/перезаливы
- Мемы с добавленной подписью/водяным знаком
- Немного обрезанные версии
- Разное JPEG-сжатие

## Выводы для новой реализации

### Вопрос: "Работает с идентичными копиями - даже если два независимых пользователя загрузили?"

**Ответ: Скорее всего НЕТ**

При независимой загрузке одного файла:

| Фактор | Влияние |
|--------|---------|
| JPEG-компрессия Telegram | Применяется при каждой загрузке, может отличаться |
| Качество исходника | Один сохранил с 80%, другой с 95% |
| Размер/разрешение | Telegram ресайзит по-разному |
| EXIF метаданные | Могут влиять на обработку |
| Кроп/редактирование | Даже минимальный crop = другой хеш |

### Что нужно для детекции "семантически одинаковых" изображений

**Perceptual hashing** (pHash, dHash, aHash) — сравнивает визуальное содержимое, а не байты:
- Устойчив к ресайзу, небольшому кропу, изменению качества
- Hamming distance для определения "похожести"
- Порог можно настраивать
